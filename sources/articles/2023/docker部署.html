IPMI_PCIE检测配置

陈可微 [IPMI PCIE] 
ipmitool.exe –I lanplus -H ip -U username -P password raw 0x30 0xce 0x00 0x16 0x05 0x00 0x00 0x00 0x05 0x00 0x01 0x00 0x00 ---关闭PCIE检测
ipmitool.exe –I lanplus -H ip -U username -P password raw 0x30 0xce 0x00 0x16 0x05 0x00 0x00 0x00 0x05 0x00 0x00 0x00 0x00 ---开启PCIE检测
ipmitool.exe –I lanplus -H ip -U username -P password raw 0x30 0xce 0x01 0x16 0x05 0x00 0x00 0x00 ---查询状态
16 05 00 00 00 05 00 01 00 00 ---返回值代表关闭PCIE检测
16 05 00 00 00 05 00 00 00 00 ---返回值代表开启PCIE检测
