Hexo反向代理配置

陈可微 [Hexo 反向代理]

Nginx配置

    location / {
        proxy_pass http://127.0.0.1:端口;
    }

    location ~ .*\.(gif|jpg|jpeg|png|bmp|swf|flv|mp3|wma|js|css)${
        proxy_pass http://127.0.0.1:端口;
        expires      30d;
    }
	
    location ~ .*\.(js|css)${
        proxy_pass http://127.0.0.1:端口;
        expires      12h;
    }
